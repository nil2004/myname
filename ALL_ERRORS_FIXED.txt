TECHNICAL DETAILS:
================================================================================

Budget Allocation API:
----------------------
- Endpoint: GET /api/budget-allocation
- Parameters: eventType, categories, totalBudget
- Returns: Budget distribution across categories
- Example: /api/budget-allocation?eventType=birthday&categories=catering,decorator&totalBudget=15000

Requests Table:
---------------
- Stores customer event requests
- Includes: name, phone, email, occasion, budget, theme, etc.
- Auto-updates timestamps
- Unique constraint on customer_phone

Vendor Media Columns:
---------------------
- image_url: Main vendor image
- banner_image: Banner for vendor card
- portfolio_images: Array of portfolio images
- portfolio_videos: Array of video URLs
- portfolio_description: Text description
- portfolio_highlights: Array of highlight points

VendorTierCard Fix:
-------------------
- Detects UUID format (contains dashes)
- Only fetches from API for real database vendors
- Uses vendor data directly for mock vendors
- Graceful error handling with fallback
