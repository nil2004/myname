================================================================================
BANNER IMAGE DISPLAY IN VENDOR RECOMMENDATIONS
================================================================================

✅ FEATURE IMPLEMENTED

The vendor cards now display banner images in the recommendations!

================================================================================
HOW IT WORKS
================================================================================

Image Priority (in order):
1. Portfolio Images (if uploaded)
2. Banner Image (if uploaded)
3. Emoji (fallback)

The system automatically:
- Shows the banner image as the card background
- Adds a dark overlay for better text visibility
- Falls back to gradient if image fails to load
- Uses emoji if no images are uploaded

================================================================================
HOW TO ADD BANNER IMAGES
================================================================================

STEP 1: Upload Banner Image in Admin Panel
-------------------------------------------
1. Go to: http://localhost:3000/admin
2. Click "Vendors Management" tab
3. Click "Edit" on any vendor
4. Scroll to "📸 Portfolio & Media" section
5. Under "🖼️ Banner Image":
   - Click "Choose File"
   - Select a high-quality image (1200x600px recommended)
   - Preview appears immediately
6. Click "Update Vendor"

STEP 2: View on Customer Site
------------------------------
1. Go to: http://localhost:3000/plan
2. Fill in the planning form
3. Click "Get AI Recommendations"
4. Vendor cards now show the uploaded banner image!

================================================================================
IMAGE SPECIFICATIONS
================================================================================

Recommended:
- Size: 1200x600px (2:1 ratio)
- Format: JPG or PNG
- Quality: High (80-90%)
- File Size: 2-5MB
- Content: Professional vendor photos, venue shots, work samples

The image will:
- Cover the top section of the card
- Have a subtle dark overlay for text readability
- Scale to fit the card dimensions
- Load quickly with proper optimization

================================================================================
TROUBLESHOOTING
================================================================================

Image not showing?
- Check if banner_image field has URL in database
- Verify image URL is accessible (open in browser)
- Check browser console for errors
- Ensure vendor is verified (only verified vendors show)
- Try clearing browser cache

Image broken?
- System automatically falls back to gradient
- Check if image URL is correct
- Verify image file exists in Supabase Storage
- Check file permissions (should be public)

================================================================================
DATABASE FIELDS USED
================================================================================

- banner_image: TEXT (single URL)
- portfolio_images: TEXT[] (array of URLs)
- image_emoji: VARCHAR(10) (fallback)

Priority: portfolio_images[0] → banner_image → image_emoji
